CloudSim:
- Create a new java project file
- Go to src file and create a package
- Right click on the package and open it in system explorer
- Copy the files given
- Close and open the project
- change the package name in the given files
- Right click on the main file click on build path
- And then click add configure build path and add the external jars files
- Run the main file as java application
- Find the file name and run it and it will show the output
